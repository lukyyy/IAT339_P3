IAT 352 Final Project

Tool Used

HTML/CSS, PHP/SQL, jQuery & AJax

Team

Kenny & Luke

Role

Coder/UX Designer

Goal

Concept and detail function design and implementing functions of website and ensure all the functions are working.


Overview

GamesBond is a game review website where public users can freely rate and comment GamesBond is a game review website where public users can rate games and share their thinking and comments about games. Like RottenTomatoes, we would like to foster game website that compares between game critics and public game players.
Through this website, we want to foster game community where any players can share their thinking, connect each other, evaluate public assessment, and compare point of view to their own views.

The website is made with PHP(function), HTML/CSS, JavaScript (design), SQL(database), and Ajax (up-to-date). Using PHP, the website can communicate with the database. This allows the website to perform better functional approach such as log-in, recommendation and more.
